# CarListener - 车辆通知监听器

## 功能说明

CarListener 是一个 Android 应用，用于监听手机短信和特定应用（默认交管12123）的通知消息。当消息中包含预设的关键字时，应用会以最大音量响铃提醒。

## 主要功能

1. **短信监听**：监听所有 incoming SMS 短信
2. **通知监听**：监听指定应用的通知（默认：交管12123）
3. **关键字匹配**：支持配置多个关键字（用逗号分隔）
4. **最大音量响铃**：检测到关键字后，以系统最大音量播放铃声并振动
5. **无网络权限**：应用不包含任何网络权限，保护用户隐私

## 权限说明

应用需要以下权限：
- `RECEIVE_SMS`：接收短信广播
- `READ_SMS`：读取短信内容
- `VIBRATE`：振动提醒
- `FOREGROUND_SERVICE`：前台服务（保持服务运行）

**注意**：应用**不包含**任何网络相关权限（如 INTERNET、ACCESS_NETWORK_STATE 等），确保数据不会外传。

## 使用方法

### 1. 安装 APK
将 `app-debug.apk` 安装到 Android 设备上。

### 2. 授予权限
打开应用后：
- 点击"授予短信权限"按钮，允许应用读取短信
- 点击"启用通知监听"按钮，在系统设置中启用通知访问权限

### 3. 配置关键字
在"短信关键字"输入框中输入需要监听的关键字，多个关键字用逗号分隔。
例如：`违章,罚款,扣分,年检`

### 4. 配置监听应用
在"监听应用包名"输入框中输入要监听的应用包名。
- 默认：`com.tmri.app.main`（交管12123）
- 如需监听其他应用，请修改为对应应用的包名

### 5. 保存配置
点击"保存配置"按钮保存设置。

### 6. 测试响铃
点击"测试响铃"按钮可以测试铃声是否正常播放。

## 工作原理

1. **短信监听**：通过 `BroadcastReceiver` 监听 `SMS_RECEIVED` 广播
2. **通知监听**：通过 `NotificationListenerService` 监听系统通知
3. **关键字匹配**：检查短信或通知内容是否包含配置的关键字（不区分大小写）
4. **响铃提醒**：匹配成功后，启动 `RingtoneService` 以最大音量播放系统铃声并振动 30 秒

## 技术细节

- **最低 Android 版本**：Android 6.0 (API 23)
- **目标 Android 版本**：Android 14 (API 36)
- **开发语言**：Kotlin
- **构建工具**：Gradle 8.13

## 注意事项

1. 首次使用需要在系统设置中手动启用"通知访问"权限
2. 某些手机厂商可能会限制后台服务，建议在电池优化设置中将本应用设为"不优化"
3. 响铃时长默认为 30 秒，可在代码中修改 `RING_DURATION` 常量
4. 应用不会收集或上传任何用户数据

## 常见问题

**Q: 为什么收不到通知提醒？**
A: 请确认已在系统设置中启用"通知访问"权限，并且监听的包名正确。

**Q: 为什么铃声不大？**
A: 应用会自动将铃声音量调至最大，但请确保手机未处于静音或振动模式。

**Q: 如何修改监听的应用？**
A: 在应用中修改"监听应用包名"字段，输入目标应用的包名即可。
